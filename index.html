Basics of the Web

The Web -
A system of extensively interlinked hypertext documents. A branch of the Internet. Invented in early 1990's. Over 30 billion pages.

HTML -
Hypertext Markup Language: The main type of document on the Web.

URL -
Uniform Resource Locator: a protocol for specifying addresses on the Internet. An address that identifies a particular file on the Internet, usually consisting of the protocol, as http, followed by the domain name.

HTTP -
Hypertext transfer protocol: the standard protocol for transferring hypertext documents on the World Wide Web.

Web Apps -
Any computer program that runs in a web browser. It is created in a browser-supported programming language (such as the combination of JavaScript, HTML and CSS) and relies on a web browser to render the application.
